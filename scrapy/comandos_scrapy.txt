CRIAR AMBIENTE VIRTUAL

- NOVA PASTA - ABRIR COM TERMINAL
PYTHON -M VENV (NOME AMBIENTE) - ENTER

- CONFIRMAR QUE O AMBIENTE FOI CRIADO
DIR - ENTER

- ATIVAR AMBIENTE
(NOME AMBIENTE)\SCRIPTS\ACTIVATE

- INSTALAR SCRAPY
PIP INSTALL SCRAPY - ENTER

- ATUALIZAR SCRAPY
PYTHON -M PIP INSTALL --UPGRADE PIP

CRIAR PROJETO SCRAPY

- ABRIR A PASTA NO VS CODE

- ATIVAR AMBIENTE VIRTUAL NO TERMINAL(CTRL + J)
.\(NOME AMBIENTE)\SCRIPTS\ACTIVATE - ENTER

- CRIAR PROJETO
SCRAPY STARTPROJECT (NOME PROJETO) - ENTER

- ENTRAR NO PROJETO
CD .\(NOME PROJETO)\ - ENTER

- CRIAR SPIDER DENTRO DA PASTA SPIDERS

- RODAR O SPIDER
CD .\(NOME PROJETO)\ - ENTER
SCRAPY CRAWL (NOME SPIDER) - ENTER

- ENTRANDO NO SITE PELO TERMINAL
SCRAPY SHELL (NOME DO SITE) - ENTER

- COMANDO PARA RODAR XPATH
RESPONSE.XPATH('//SPAN[@CLASS="TEXT"]/TEXT()').GET() - ENTER - PRIMEIRO RESULTADO
RESPONSE.XPATH('//SPAN[@CLASS="TEXT"]/TEXT()').GETALL() - ENTER - TODOS OS RESULTADOS

- SAIR
CTRL + Z

- BAIXAR DADOS EM CSV
SCRAPY CRAWL (NOME DO SPIDER) -O DADOS.CSV

- TESTAR O XPATH NO CONSOLE DO NAVEGADOR
$x("XPATH")

- BAIXAR DADOS EM JSON E XML
-O para sobrescrever arquivos existentes
-o para acrescentar dados a arquivos existentes(n√£o funciona com json)

scrapy crawl nomebot -O dados.csv
scrapy crawl nomebot -O dados.xml
scrapy crawl nomebot -O dados.json
